# Rapport SSI — Semaine

Application React/Vite pour saisir les informations SSI du mardi au vendredi.

## Fonctions

- État du SSI à l’arrivée et au départ
- Saisie libre des travaux réalisés
- Ajout d’essais ZA / ZC / ZF
- Position PS / DPS
- Observations et anomalies
- Génération automatique d’un compte rendu
- Copie du texte
- Sauvegarde automatique dans le navigateur avec `localStorage`

## Lancer le projet

```bash
npm install
npm run dev
```

## Déployer sur Vercel

1. Importer le dossier sur GitHub.
2. Dans Vercel, cliquer sur **Add New Project**.
3. Sélectionner le dépôt GitHub.
4. Vercel détecte automatiquement Vite.
5. Cliquer sur **Deploy**.

Les données sont enregistrées uniquement dans le navigateur utilisé.
